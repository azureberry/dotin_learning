VagrantTest
=====


## Requirements

You'll need to have the following items installed before continuing.

  * [vagrant](https://www.vagrantup.com/)
  * [virtualbox](https://www.virtualbox.org/)
  * [chefDK](https://downloads.chef.io/chef-dk/)


## Quickstart
  (windows gitbash)

  ```
  cd vagrant
  (vagrant box update)
  vagrant up
  vagrant ssh
  (vagrant reload --provision)
  ```

(serverspec test)

  ```
  cd /vagrant/
  rake
  ```
